# Source links — clickable URLs for the bench

Free = no login. Paid = subscription. Data = pullable numbers. The first block is what verifies the Excel table.

## ① Free verified data — pull these to upgrade the table (W → V)
- **Eurostat — emergency oil stocks, days equivalent** (DE, FR, IT, ES, NL, PL, monthly): https://ec.europa.eu/eurostat/databrowser/view/nrg_stk_oem/default/table
  - JSON API: https://ec.europa.eu/eurostat/api/dissemination/statistics/1.0/data/nrg_stk_oem?format=JSON&lang=EN
- **Eurostat — oil stocks in tonnes** (`nrg_stk_oilm`): https://ec.europa.eu/eurostat/databrowser/view/nrg_stk_oilm/default/table
- **JODI-Oil World Database** (India, Thailand closing stocks; CSV): https://www.jodidata.org/oil/database/data-downloads.aspx
- **EIA Open Data** (free API key; SPR `WCSSTUS1`, Brent `RBRTE`, commercial crude `WCESTUS1`): https://www.eia.gov/opendata/
- **EIA Weekly Petroleum Status Report** (US SPR + commercial, Wednesdays): https://www.eia.gov/petroleum/supply/weekly/
- **IEA — Oil stocks of IEA countries** (days of net-import cover; some free, full = OMR sub): https://www.iea.org/data-and-statistics/data-tools/oil-stocks-of-iea-countries
- **IMF PortWatch** (Hormuz transit, free): https://portwatch.imf.org/
